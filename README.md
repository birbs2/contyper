# Contyper

Contyper is a program that makes it really easy for conlangers to type any characters you might want. It is my attempt at fixing one of my biggest issues while conlanging: Special characters.

I always found it so annoying to copy and paste any character you might want from google. Opening up my google doc with all of my special characters or reffering to my charts to copy and paste always felt so clunky to me. 

## Design Principles
The main design principles of contyper are as follows:
 * Contyper should not intefere with normal keyboard functions. You should be able to forget about Contyper until you need it again.
 * Contyper should be fully customizable. All features should be optional and be toggleable. On top of that, any configuration file/ruleset should be easily written and easily read.
 * Contyper should be easy to use. All aspects of the program should be obvious and/or be documented to someone who hasn't used the program before.
 * Contyper should provide tools for as many use cases as possible.
